Present: Buddy Allocator (TAMU Project - CSCE 313)

As part of an Operating Systems class I am currently taking at TAMU, I am creating a buddy allcoator. Often the OS functions that manage memory are too generic, and many programs have specific memory request patterns. Furthermore, many high-performance applications become bottlenecked. That is why many applications use their own memory manager to reduce load on the OS memory manager. This project is essentially creating a memory managing program using C++ to implement it. So far, the project has reenforced in my mind the importance of attempting not to waste memory.
Check out my github to learn more about this current project.
